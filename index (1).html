function updateServiceWidth() { if (window.innerWidth > 768) { serviceItems.forEach(item => { // در صورتی که نیاز باشد عرض دقیق تری اعمال شود، اینجا انجام می‌شود // اما در این طرح، flex: 0 0 auto کافی است. }); } } window.addEventListener('resize', updateServiceWidth); updateServiceWidth(); // اجرای اولیه }); راهنمای پیاده‌سازی و ساخت فایل DOCX برای تکمیل پروژه و ایجاد سند نهایی DOCX، مراحل زیر را دنبال کنید: ۱. ساختار پوشه‌ها یک ساختار پروژه به شکل زیر ایجاد کنید: LegalWebsite/ ├── index.html (محتوای بخش ۱) ├── style.css (محتوای بخش ۲) ├── script.js (محتوای بخش ۳) └── images/ ├── logo.png (لوگوی موسسه) └── hero-bg.jpg (تصویر پس‌زمینه بخش Hero) ۲. جایگزینی محتوای پیش‌فرض تصاویر: مطمئن شوید که logo.png و hero-bg.jpg در مسیرهای صحیح قرار گرفته‌اند. در index.html، مسیرهای "images/logo.png" و "../images/hero-bg.jpg" (بسته به محل قرارگیری فایل) باید دقیق باشند. لینک‌ها: لینک‌های شبکه‌های اجتماعی در فوتر ( تگ‌ها) را با آدرس‌های واقعی موسسه خود جایگزین کنید. نقشه: کد
